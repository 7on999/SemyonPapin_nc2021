Элементы для форматирования текста:

Полужирное начертания шрифта можно достигнуть использовнием тэга 'i'
Тэг 'strong' также задаёт полужирное начертание шрифта, но при этом относится к тегам логической разметки, указывая браузеру на важность текста
Тэг 'em' отображает шрифт курсивом, придавая тексту значимость.
Тэг 'i' отображает шрифт курсивом, НЕ придавая тексту значимость.
Тэг sub задает нижний индекс

Тэг sup задает верхнй индекс - используется для обознчения степеней
Тэг del зачеркивает
Тэг ins подчеркивает
Тэг mark Применяется для выделения фрагментов текста в справочных целях, окрашивая блок символов желтым цветом.

Элементы для ввода «компьютерного» текста:

Тэг code cлужит для выделения фрагментов программного кода. Отображает текст моноширинным шрифтом.
Тэг kbd Отмечает фрагмент как вводимый пользователем с клавиатуры. Отображает текст моноширинным шрифтом.

Элементы для оформления цитат и определений

Тэг blockquote выделяет цитаты внутри документа, выделяя его отступами и переносами строк.

Тэг bdo используется для изменения текущего направления текста
Тэг q используется для выделения коротких цитат. Браузерами заключается в кавычки
Тэг cite применяется для выделения цитат, названий произведений, сносок на другие документы.
Тэг cite позволяет выделить текст как определение. Несмотря на наличие данного элемента, рекомендуется выделять текст силами CSS..

Абзацы, средства переноса текста

Тэг p разбивает текст на отдельные абзацы, отделяя друг от друга пустой строкой. Браузер автоматически добавляет верхнее и нижнее внешнее поле margin, равное 1em, при этом поля соседних абзацев «схлопываются».

Тэг 'br' переносит текст на следующую строку, создавая разрыв строки.
